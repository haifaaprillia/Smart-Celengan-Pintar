# Smart-Celengan-Pintar

Raw Data Penelitian

Repository ini berisi kumpulan raw data yang digunakan dalam kegiatan penelitian Celengan Pintar. Data disimpan dalam bentuk data mentah tanpa dilakukan pengolahan atau analisis lebih lanjut.

Data yang tersedia meliputi hasil wawancara responden terkait kebiasaan menabung, log percakapan Telegram Bot yang digunakan dalam sistem Celengan Pintar, daftar fitur Telegram Bot, serta screenshot percakapan Telegram Bot sebagai dokumentasi pendukung.

Seluruh data dalam repository ini digunakan sebagai bagian dari proses penelitian dan dikumpulkan untuk keperluan akademik.
